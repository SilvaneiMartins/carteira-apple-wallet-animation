## Executar o game

```bash
    # Clone o Repositório:
    $ git clone https://github.com/SilvaneiMartins/carteira-apple-wallet-animation

    # Entre no Diretório:
    cd carteira-apple-wallet-animation

    # Executar o projeto
    $ expo start
    ou
    $ npx expo
```
